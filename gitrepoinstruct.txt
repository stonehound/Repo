# How to initialize a Git Repository:
Firstly make a directory on your computer, mkdir Project Name.
Create a README.md file, touch README.md.
Initiate the git repository, git init.
This will create all metadata needed to function as a repo, ls .git -a.

Example:
mkdir ProjectName
touch README.md
vim README.md
git init
ls .git -a


# How to add a file to the repository:
To add a file, git add File Name.
To add all files, git add --all.
To check on the status of of the repository, git status.

Example:
git add Filename.txt
git status


# How to commit changes to the repository:
To commit changes, git commit -am "Your message."
